


# Decision Tree Regression Project

## Project Overview
This project demonstrates the implementation of the Decision Tree Regression algorithm using Python and Streamlit.

The model predicts continuous numerical values based on input features using a tree-based regression approach.

---

## Machine Learning Type
- Supervised Learning
- Regression
- Decision Tree Regressor

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib

---

## Dataset
The project uses a regression dataset for prediction.

Example:
- Salary Prediction
- House Price Prediction
- Student Marks Prediction

---

## Features Used
Example features:
- Experience
- Study Hours
- Area
- Age

---

## Target Variable
The target variable contains continuous numerical values such as:
- Salary
- Price
- Marks

---

## Algorithm Used
```python
DecisionTreeRegressor()
```

Decision Tree Regression predicts numerical values by creating decision rules in a tree structure.
